# Non-Graphic Dead-Cat Outcome

Catbox will use a Dead-Cat Outcome for the Schrodinger-inspired branch, but the outcome must remain non-graphic. The image should make the cat's state recognizable without blood, visible injury, mutilation, gore, or horror spectacle.

**Considered Options**

- Keep the Absent-Cat Outcome as symbolic absence.
- Use a non-graphic Dead-Cat Outcome.
- Use a graphic animal-death outcome.

**Consequences**

The experience becomes closer to the Schrodinger's cat premise while avoiding graphic animal death. Outcome prompts, UI language, validation docs, and tests should use the Dead-Cat Outcome term, and generation prompts should actively avoid graphic injury details.
