# Single-Branch Captured Denoising Trace

Catbox will show a Captured Denoising Trace from the selected diffusion path when the Model Backend can capture intermediate frames. Catbox will not generate both living and dead branches just to display two possible traces before revealing one.

**Considered Options**

- Show theatrical Observation Noise only.
- Capture and display the selected branch's real denoising frames.
- Generate both branches and display dual-branch denoising theater before revealing one.

**Consequences**

The waiting experience becomes more truthful and more visually connected to the real model run. Runtime remains focused on one Generated Outcome instead of doubling work for an unchosen branch. Browser UI copy must distinguish captured model frames from fallback Observation Noise, and the trace remains an Ephemeral Outcome rather than saved history.
